# EcdlManager
Program to manage status exams. Allows you to create exams, examinations, skillcards and much more!

## Getting Started
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes

### Prerequisites

* VisualStudio - The editor for programming
* Mysql - Dependency Management
* WorkBench - Used to generate and manage the DataBase

### Build With

* C# - Object-oriented programming language developed by Microsoft
* Mysql - Database Management System
